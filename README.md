# Student Grade Tracker

A Python program that models a simple student grade tracking system. Built to demonstrate core Python concepts through a practical, real-world example.

---

## What It Does

- Creates student records with names, emails, and grades
- Adds and removes grades from each student's record
- Calculates average grades
- Looks up students by email using a dictionary
- Collects unique grades across all students using a set
- Locks grade snapshots using immutable tuples
- Validates email addresses using regular expressions
- Counts how many grades across all students are above 90

---

## How to Run

No external dependencies required. Just Python 3.

```bash
python student_tracker.py
```
